# ONG Rio Limpo — Projeto Jovem Tech

Este repositório contém um site estático (HTML, CSS e JS) com **3 páginas**:
- `index.html` (Início)
- `sobre.html` (Sobre)
- `contato.html` (Contato)

## Observações para avaliação
- O código contém comentários com a palavra **ALTERAÇÃO** indicando mudanças autorais.
- Substitua as imagens em `assets/img/` por fotos relacionadas à ONG (reais ou de banco com licença).
- Todos os arquivos estão em uma **única pasta** (esta).
- Para projeto real: preencha o bloco de assinatura em `sobre.html` com nome, cargo e assinatura do responsável.
- Publicação: pode usar GitHub Pages, Netlify ou Vercel. Basta subir estes arquivos.

## Como visualizar localmente
Abra `index.html` no navegador.
